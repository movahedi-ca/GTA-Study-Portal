# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-19
### Added
- Initial release of the GTA College Programs Guide.
- Comprehensive database of 17 vetted two-year, PGWP-eligible programs spanning tech, health, engineering, early childhood, and supply chain.
- Semantic categorization of CIP codes.
- Implementation of light and dark mode aesthetic themes.
- Client-side text-index search implementation alongside dropdown selectors.
- Fully rewritten descriptions, admission criteria, and career layouts extracted from research file matrices.
- "Excluded Appendix" cataloging Centennial College programs and non-target items.
